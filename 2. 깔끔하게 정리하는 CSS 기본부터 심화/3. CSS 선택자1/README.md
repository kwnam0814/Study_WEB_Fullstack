# CSS

## 기본 선택자

- 전체

  - 문서 전체를 선택하는 선택자

  ```css
  * {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
  }
  ```

- 태그

  - 태그명으로 선택

- id

  - id 속성값으로 선택

- class

  - class 속성값으로 선택

- 특성(attribute)
  - [속성]
  - [속성=속성값]
  - 위와 같은 형식을 선택

## CSS 우선순위

- `id > class = attribute > tag`
