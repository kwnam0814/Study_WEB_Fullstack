# JavaScript

## 반복문

### 함수

함수란 특별한 목적의 작업을 수행하도록 설계된 독립적인 코드 블록이다.

함수는 필요할 때마다 호출하여 해당 작업을 반복하여 수행할 수 있다.

변수가 데이터를 담기 위한 그릇이었다면, 함수는 소스코드를 담기 위한 그릇이다.

마치 리모컨처럼 리모컨의 원리는 다 몰라도 버튼을 누르면 채널이 이동하죠? 그 버튼을 우리는 코딩에선 함수라고 한다.

**함수를 정의하는 방법**

```javascript
// 기본형
// 호이스팅 지원
function 함수명() {
  소스코드;
}

// 익명함수
// 호이스팅 불가
let 변수명 = function () {
  // 함수명이 없어서 익명함수라 불림
  소스코드;
};

// ES6 화살표 함수
// 익명함수의 일종
// this 바인딩 관련 간단 설명
let 변수명 = () => {
  소스코드;
};
```

`💡함수 정의 방식.. 호이스팅?`

소스코드는 위에서부터 아래로 흐른다.

기본형 함수는 함수 호출을 선언 전에 해도 문제가 없다.

호이스팅(끌어올리다)되기 때문이다.

반면 익명함수는 호이스팅을 지원하지 않아 선언 전에 호출을 하면 에러가 나게 된다!

**함수를 호출하는 방법**

```javascript
// 기본형 함수 호출
함수명();

// 익명함수 호출
변수명();
```

#### 더하기 함수 예제

```javascript
function basic(num1, num2) {
  console.log(num1 + num2);
}

basic(1, 3);
```

#### 파라미터

함수 예제를 다시 보면 괄호안에 `num1, num2` 라는 변수가 있는 것을 확인할 수 있다.

이들은 매개변수(parameter) 이다.

함수의 재료가 되는 요소이다.

#### 리턴

리턴값을 통해 함수에서 실행된 결과를 전달 받을 수 있다.

리턴은 함수의 실행을 중단하고 값을 반환한다.

```javascript
function basic(num1, num2) {
  return num1 + num2;
}

let result = basic(1, 3);

console.log(result);
```
