# JavaScript

## 변수와 자료형

### 변수

#### let과 const 차이

`재할당 가능 여부`

```javascript
let myName = "홍길동";
console.log(myName);

myName = "김철수";
console.log(myName);
```

```javascript
const myName = "홍길동";
console.log(myName);

myName = "김철수"; // 재할당 불가 에러
console.log(myName);
```

`const`

- 재할당이 필요없는 상수, 객체에 사용하는 변수 선언방식
- 프로그래밍 도중 의도치 않게 발생하는 재할당을 방지하기 위해서 기본적으로 변수 선언에는 `const` 를 사용하는 것이 안전!

### 연산자와 조건문

#### 연산자

- 컴퓨터 내부에서 다양한 계산을 하는 기호를 의미함.
- 덧셈, 뺄셈, 비교 등.. 다양한 연산자가 존재함.

#### 산술 연산자

| 종류 | 의미   | 예시   |
| ---- | ------ | ------ |
| +    | 더하기 | A + B  |
| -    | 빼기   | A - B  |
| \*   | 곱하기 | A \* B |
| /    | 나누기 | A / B  |
| %    | 나머지 | A % B  |

#### 비교 연산자

| 종류 | 의미                                                   | 예시    |
| ---- | ------------------------------------------------------ | ------- |
| >    | 크다(작다)                                             | A > B   |
| >=   | 크거나 같다(작거나 같다)                               | A >= B  |
| ===  | 같다(strict) - 값 뿐만 아니라 데이터 타입도 검사함     | A === B |
| !==  | 다르다(strict) - 값 뿐만 아니라 데이터 타입도 검사함   | A !== B |
| ==   | 같다 - 데이터 타입은 보지 않고 그저 값만 같으면 true   |         |
| !=   | 다르다 - 테이터 타입은 보지 않고 그저 값만 다르면 true |         |

#### 논리 연산자

| 종류 | 설명                                                      |
| ---- | --------------------------------------------------------- |
| \|\| | or 연산자. 피연산자 중 값이 하나라도 true라면 true를 반환 |
| &&   | and 연산자. 피연산자 모두 true여야 true를 반환            |
| !    | not 연산자. 논리의 반댓값을 반환                          |

### 중간정리 실습 : 적정체중 계산 코딩

`적정체중 = (키 - 100) * 0.9` 라고 합니다.

1. 본인의 키를 변수에 저장하세요.
2. 본인의 적정체중을 변수에 저장하세요
3. 아래 문자열을 콘솔창에 반환하세요.

`당신의 키는 -cm이며 적정체중은 -kg입니다.`

```javascript
const myHeight = 175;
let goodWeight = (myHeight - 100) * 0.9;

console.log(goodWeight);
console.log(
  "당신의 키는 ",
  myHeight,
  "cm이며, 적정체중은 ",
  goodWeight,
  "kg입니다.",
);
```
