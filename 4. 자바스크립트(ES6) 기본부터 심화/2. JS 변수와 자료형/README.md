# Javascript

## 변수와 자료형

`사전지식`

**자바스크립트 사용 방법**

```javascript
<script>js코드</script>
```

- JS를 사용하기 위해서는 body태그 가장 하단에 script 태그에 작성.
- 직접 js코드를 작성할 수 있으며 파일을 분리해서 사용할 수도 있다.

**코드 작성 시 주의사항**

1. 코드는 대소문자를 구분하여 작성
2. 코드 한 줄을 작성한 뒤엔 세미콜론(;)을 사용 → 코드 의미를 구분하는 기호
3. 문자형 데이터를 작성할 때는 따옴표 겹침을 주의!
4. 코드를 작성할때는 괄호의 짝이 맞아야 함

**자바스크립트 주석은 `//` 으로 표시!**

### 변수

변수란, 데이터를 담을 수 있는 박스(메모리 공간)이다.

변수에 저장할 수 있는 데이터는 문자(String), 숫자(Number), 논리형(Boolean), Null, 객체(Object) 데이터가 있다.

#### 변수 선언

- 변수를 선언할 때는 **var**, **let**, **const** 키워드를 변수명 앞에 붙여서 사용함.
  - var : 옛날 자바스크립트 느낌 -> 요즘에는 안쓰려고 노력
- 변수명에는 한글을 사용할 수 없다.
- 변수명은 의미에 맞게 만드는 것이 좋다.
- 변수명을 지정하는 네이밍 컨벤션은 카멜, 스네이크 표기법 등이 있다.

```javascript
let myVariable; // 카멜 케이스
let my_variable; // 스네이크 케이스
```

변수를 선언하면 데이터를 저장할 수 있는 메모리 공간이 할당 된다.

box라는 변수에 10이라는 숫자형 데이터를 대입해보자.

초기에는 비어 있는 null 상태이다.

```javascript
let box;
box = 10;
console.log(box);
```

변수 하나에는 하나의 값만 넣을 수 있다.

변수 box에 20이라는 새로운 숫자를 대입하면 어떻게 될까?

```javascript
box = 20;
console.log(box);
```

기존에 있는 것은 삭제되고 20이 그 자리를 대체하게 된다.

#### 데이터 타입

변수에는 다양한 자료형(데이터 타입)을 저장할 수 있다.

문자형, 숫자형, 논리형 등이 있다.

1. **문자형** : 문자로 표현하고 싶은 데이터는 따옴표로 감싸서 표현.

```javascript
let str = "js"; // 큰따옴표
let str2 = "js"; // 작은따옴표
```

2. **숫자형**

```javascript
let num = 1;
```

3. **논리형**

데이터는 참 또는 거짓으로 분류할 수 있다.

예를들어 "10은 100보다 크다"라는 데이터는 false 데이터!

"100은 10보다 크다"라는 데이터는 true 데이터!

```javascript
let bool = true;
let t = 100 > 10;
```

4. **배열, JSON**

- typeOf 메서드
  - 해당 데이터의 자료형을 알고싶을 때 사용하는 기능.

```javascript
typeof 데이터 또는 변수;
```

#### 변수 선언 시 주의사항

1. 변수명 첫글자는 영문자, $, \_(언더바)만 올 수 있다.
2. 변수명으로는 예약어는 사용 불가능하다. 예약어는 자바스크립트 내부에서 이미 사용중인 단어를 의미한다.
3. 변수명은 최대한 해당 데이터의 의미를 잘 포함하게 작성한다.
4. 대소문자를 구분해야 한다.
