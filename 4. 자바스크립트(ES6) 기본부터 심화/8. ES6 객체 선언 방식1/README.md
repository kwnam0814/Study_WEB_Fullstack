# JavaScript

## 객체 선언 방식1

### 객체지향프로그래밍 (OOP)

OOP는 프로그래밍을 어떻게 할 것인가, 방법론 중 하나이다.

객체를 사용해서 모델링하고 프로그래밍을 더 효율적으로 할 수 있게 만들어 주는 방법론이다.

하지만 객체지향프로그래밍이란 단어는 굉장히 생소하다.

- 클래스 : 사물이나 개념 등의 컨셉 (붕어빵 틀)

- 객체 : 클래스에 맞춰서 만들어진 객체 (붕어빵)

- 속성 : 객체가 가지는 데이터 (자동차 바퀴 갯수, 승차 인원, 창문 갯수 등..)

- 메서드(함수) : 객체가 가지는 기능 (라이트 켜기, 가속하기, 감속하기, 트렁크 열기 등..)

**객체 선언 방식**

- 리터럴 방식
- 클래스 방식
- 생성자 방식 new obj() (권장하지 않음!)
- 함수 방식 (권장하지 않음!)

### 1. 리터럴 방식

다루는 방식은 앞서 컨테이너 자료형 파트에서 배운 json과 같다.

사실 json을 다루는 것을 객체 리터럴 방식으로 한 것이다.

```javascript
const userInfo = {
  name: "홍길동",
  birth: "2020-10-10",
  getName: function () {
    return this.name; // this : 메서드가 정의된 객체를 가리키는 말
  },
};

console.log(userInfo.name);
```

`this와 화살표 함수`

```javascript
const obj = {
  nickname: "하하",
  age: 22,
  say: function () {
    console.log(this);
    console.log(`제 이름은 ${this.nickname}, 나이는 ${this.age}살 입니다.`);
  },
};

obj.say();
```

**객체 리터럴 방식 예제**

1. 리터럴 방식으로 빈 객체를 생성.
2. 두 개의 숫자를 속성으로 설정하는 메서드를 작성. (setNumbers)
3. 속성으로 설정된 두 숫자를 더한 결과를 리턴하는 메서드, 곱한 결과를 리턴하는 메서드 작성. (sum, multiply)
4. setNumbers로 두 숫자를 속성으로 설정하고 sum, multiply 메서드를 호출.
