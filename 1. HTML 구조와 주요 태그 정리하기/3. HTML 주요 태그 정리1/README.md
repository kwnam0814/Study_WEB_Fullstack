# HTML

## 주요 태그 정리

### div

```html
<div></div>
```

- 네모난 박스라고 생각하면 됨.
- 대부분의 레이아웃은 div태그로 구성.
- css, js를 특정 부분에 제어할 때 유용.
- class, id 속성을 함께 사용하는 것이 일반적임.
- 전체 너비 차지
- 높이는 그 안에 들어있는 만큼 차지

### h1 ~ h6

```html
<h1>안녕</h1>
```

- 제목이나 강조하고 싶은 부분을 나타내는 태그.
- h1의 폰트 사이즈가 가장 크고, 숫자가 커질수록 사이즈가 작아진다.

### img

```html
<img src="" alt="" />
```

- 이미지를 보여줄 수 있는 기능을 하는 태그.
- 닫는 태그가 없음.
- src 속성 : 이미지의 소스는 어디에 있는가?
- alt 속성 : 네트워크 등 여러 가지 이슈로 소스 안의 사진을 제대로 가져오지 못할 때, alt 속성 내부의 텍스트를 대신 띄어줌.

### input

```html
<input type="" />
```

- 사용자의 입력을 받는 태그.
- 자주 사용하는 type 속성값
  - text
  - button
  - checkbox
  - password
  - number
  - radio
