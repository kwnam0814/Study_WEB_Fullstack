# HTML

## 크로스 브라우징, 공식문서 활용방법

### 크로스 브라우징 ?

- 서로 다른 웹 브라우저에서도 동일한 UX/UI를 구현하기 위한 기술.
- 웹 브라우저마다 버전별로 지원하지 않는 기술이 존재할 수 있다.
- 이런것을 해결하고 모든 브라우저에서 정상적으로 동작하게 하는 작업이 바로 크로스 브라우징이다.
- 크로스 브라우징 이슈는 인터넷 익스플로러에서 가장 많이 발생했다.

- [Can I use: 브라우저 호환성 검사](https://caniuse.com/)
